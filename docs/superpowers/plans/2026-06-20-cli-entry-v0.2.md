# MiniCode CLI Entry v0.2 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add an installable `minicode` console command with argparse while preserving the legacy script entry and v0.1.1 behavior.

**Architecture:** Centralize CLI behavior in `minicode_cli.py`, keep `main.py` as a thin wrapper, and use setuptools project scripts for installation. The CLI delegates agent execution to the existing `QueryLoop`.

**Tech Stack:** Python standard library, argparse, setuptools, pyproject.toml, unittest.
